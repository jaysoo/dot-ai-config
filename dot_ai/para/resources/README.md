# Resources

Reference materials collected for future use. Topics of interest and learning materials.

## Purpose

Resources are information you want to keep for reference:
- Architecture documentation for repositories
- Utility scripts for common tasks
- How-to guides and reference materials

## Current Resources

| Resource | Folder | Description |
|----------|--------|-------------|
| Architectures | `architectures/` | Repository architecture documentation |
| Scripts | `scripts/` | Utility scripts and automation |

## Adding New Resources

1. Create a new folder: `resources/[resource-name]/`
2. Add `README.md` describing the resource collection
3. Update this README with the new resource
