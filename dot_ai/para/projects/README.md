# Projects

Short-term efforts with specific goals and end dates.

## Purpose

Projects are time-bound work with clear completion criteria:
- Feature implementations
- Migrations and refactors
- RFCs and proposals
- Initiatives with deadlines

## Current Projects

- **[nx-import-vite](nx-import-vite/)** — NXA-1075: Skill reference for importing Vite projects into Nx workspaces

## Adding New Projects

1. Create a new folder: `projects/[project-name]/`
2. Add `README.md` with:
   - Goal and scope
   - Timeline/deadline
   - Key milestones
   - Related resources

## Completing Projects

When a project is complete:
1. Move folder to `../archive/projects/[project-name]/`
2. Add completion note at top of README
3. Update any references in CLAUDE.md if needed
