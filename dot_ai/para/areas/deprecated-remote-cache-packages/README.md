# Deprecated Remote Cache Packages

Tracking weekly npm download counts for the old `@nx/*-cache` packages and their `@nx/powerpack-*` predecessors. These packages have been superseded by Nx Cloud's built-in remote caching and are candidates for eventual deprecation/EOL.

**Packages tracked:**
- [`@nx/s3-cache`](https://www.npmjs.com/package/@nx/s3-cache)
- [`@nx/gcs-cache`](https://www.npmjs.com/package/@nx/gcs-cache)
- [`@nx/shared-fs-cache`](https://www.npmjs.com/package/@nx/shared-fs-cache)
- [`@nx/azure-cache`](https://www.npmjs.com/package/@nx/azure-cache)
- [`@nx/powerpack-s3-cache`](https://www.npmjs.com/package/@nx/powerpack-s3-cache)
- [`@nx/powerpack-gcs-cache`](https://www.npmjs.com/package/@nx/powerpack-gcs-cache)
- [`@nx/powerpack-shared-fs-cache`](https://www.npmjs.com/package/@nx/powerpack-shared-fs-cache)
- [`@nx/powerpack-azure-cache`](https://www.npmjs.com/package/@nx/powerpack-azure-cache)
