## Cross-Cutting Themes

### 1. Rolldown 1.0 stable resets the tooling conversation
Rolldown shipped 1.0.0 (May 7) and has been Vite 8's bundler since March. Semver API is locked. A standalone `@nx/rolldown` plugin is now justified -- deferring further means falling behind the ecosystem. OXC toolchain (now at 0.132.0) is co-evolving rapidly.

### 2. Angular 22 RC is the dominant near-term risk
Two RC drops in one week (rc.0 May 13, rc.1 May 20) signals the team is close. GA window is ~1-2 weeks. All cumulative breaking surface is locked; RC added new security guards (SVG/script injection, strict `TitleStrategy` typing). `@nx/angular` prep must be complete before GA.

### 3. Next.js security surge
16.2.5/16.2.6 together patch 12 CVEs. This is the largest Next.js security event in recent memory. Nx template pinning and user-facing guidance are overdue.

### 4. webpack moving toward zero-config
5.107.0 adds experimental TypeScript (Node.js type-stripping) and HTML modules. The long-term arc: build a web app with webpack and no external loaders. Nx generators built around ts-loader and html-webpack-plugin should start planning migration paths.

### 5. Rspack v2 is actively shipping features
Three patches in 3 weeks (2.0.2-2.0.4) add meaningful features (CSS global modules, `css/global` type, namespace tree-shaking). The `@nx/rspack` v2 integration gap is widening with each release.

### 6. Quiet tools deserve a note
esbuild (0.28.0), TypeScript (6.0.3), and webpack's patch-level activity (5.107.2 perf) remain stable. The ecosystem is bifurcated: frameworks and emerging bundlers are active; established compilers are quiet.
