**Current stable:** 21.2.11 (2026-04-29) | **Next major preview:** 22.0.0-next.10 (2026-04-29)

**Releases in window:**
- v21.2.6 → v21.2.11 (six stable patches)

**Angular 22.0.0-next.10 highlights (new this month):**
- `injectAsync` helper for DI
- Optional chaining now returns `undefined` correctly in expressions
- Model and output migrations added
- Compiler: let-declaration span fix
- Core: view-queries metadata ordering fix in JIT mode; non-object event guards; prevent event-replay double-invocation during hydration

**Cumulative Angular 22 breaking surface (unchanged from April):**
- **Node 20 dropped** (min Node 22.22.0 / 24.13.1)
- **TypeScript 5.9 dropped** — TS 6+ required (next.7)
- `FetchBackend` is default HTTP backend; `withFetch()` deprecated no-op; migration via `withXhr()`
- `provideRoutes()` removed
- `ComponentFactoryResolver` / `ComponentFactory` removed (next.7)
- `paramsInheritanceStrategy` default flipped `'emptyOnly'` → `'always'` (next.9)
- `appRef.bootstrap` second arg no longer accepts `any`
- Compiler throws on duplicate input/output bindings, `data-` prefixed binding attempts
- `nullishCoalescingNotNullable` / `optionalChainNotNullable` diagnostics

**Angular 22 cumulative new APIs:**
- `injectAsync` helper (next.10) — NEW THIS MONTH
- `@Service` decorator (next.9) — new DI primitive
- Incremental hydration default-on (next.8)
- Shadow root bootstrap (next.8)
- In-page AI tools (next.8)
- `bootstrapApplication` via `ApplicationRef` config (next.7)

**@nx/angular impact:** P0. Same as April but now urgent — RC/GA window approaching. Need: Node 22+ / TS 6 gating, FetchBackend migration, router default-strategy doc + migration, `ComponentFactoryResolver` sweep, `injectAsync` + `@Service` generator option follow-up.

## Cross-Cutting Themes

### 1. Stability after April's churn
Vite, esbuild, webpack, React, TypeScript, and Vue all shipped nothing in May. This contrasts sharply with April (Rspack 2 GA, Vite 8 patch storm, esbuild 0.28). The ecosystem is digesting April's breaking changes — a good window for Nx to integrate.

### 2. Angular 22 RC/GA window opening
Seven preview drops since March 19 (next.4 → next.10). Pattern suggests RC within ~2 weeks, GA within ~4 weeks. `@nx/angular` prep needs to be in flight now.

### 3. Remix 3 alpha → beta is a real signal
Beta promotion + final-ish package layout means React Router 7 isn't the only React routing story `@nx/remix` needs to consider. Remix 3 stable could land alongside or shortly after Angular 22 GA.

### 4. Next.js 16.3 canary is a sleeper
10 canaries in 16 days. `unstable_io` graduation is the headline; Turbopack perf work is incremental. `@nx/next` should plan for a 16.3 GA in 4-8 weeks.

### 5. Rolldown's quiet defaults change
`inlineConst.mode = 'smart'` default is a subtle behavior shift that won't show up in most error reports but may produce different bundle output. Easy to miss; deserves a snapshot test.

### 6. Node 18/20 deprecation continues
Angular 22 dropped Node 20. Rspack 2.0 dropped Node 18. Nx baseline should plan Node 22 floor for next major.
