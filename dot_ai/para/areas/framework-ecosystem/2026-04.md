**Action:** Yes — start on Nx 22-aligned Angular 22 plugin branch now. TS 6 is the biggest new constraint this month.

## Cross-Cutting Themes

### 1. Post-GA stabilization churn
Vite 8 shipped Mar 12 with 8 subsequent patches in one month; Rspack 2 hit RC; Angular 22 next.7 is removing APIs. All three signal that the frameworks flagged last month as "coming soon" are now in the critical window. Nx plugin work is **more urgent**, not less.

### 2. Security backporting
Next.js CVE-2026-23869 backported to both 16.x and 15.x LTS. React shipped simultaneous 19.0/19.1/19.2 patches. Nx should have a clear runbook for "emergency floor bump" migrations when upstream CVEs land.

### 3. TypeScript 6 is now mandatory for bleeding-edge Angular
Angular 22 drops TS 5.9. Nx workspaces targeting Angular 22 will need TS 6 bundled. This intersects with Nx's own TS version planning.

### 4. Rolldown / Vite coupling
The Vite 8 patch cadence is really Rolldown RC cadence. Until Rolldown hits 1.0 stable (still rc.15), expect ongoing `@nx/vite` user reports about Rollup-plugin edge cases.

### 5. Bundler tree-shaking arms race
Rspack 2 shipped side-effect-free function call analysis; webpack 5.106 added CJS destructuring tree-shaking. Bundlers are converging on more aggressive optimization defaults. Nx generators should validate these don't regress user output.
