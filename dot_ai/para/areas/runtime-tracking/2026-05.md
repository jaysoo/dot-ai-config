## Bun

### Release Summary (since 2026-03-05)

| Version | Date | Key Changes |
|---------|------|-------------|
| **v1.3.13** | 2026-04-20 | **`bun test --parallel/--isolate/--shard/--changed`**, install streams tarballs (17x less memory), 5.5x faster gzip (zlib-ng), source maps 8x less memory, range requests in `Bun.serve()`, SHA3 in `node:crypto`, `ws+unix://`, 82 issue fixes |
| v1.3.12 | 2026-04-10 | Domain migration to bun.com, 8 contributors |

### Major Observation: Bun 1.3.13 ships test orchestration

`bun test --parallel`, `--isolate`, `--shard`, `--changed` is **direct overlap with Nx affected/parallel orchestration**. Bun is moving from runtime to monorepo-test-runner territory. Worth flagging to Nx product:
- `--changed` mirrors `nx affected --target=test`
- `--shard` mirrors Nx Cloud distributed task execution at the test level
- `--isolate` is process-level test isolation (Nx delegates this to the runner today)

Not an immediate threat — Bun's monorepo workspace story still has holes (lockfile parsing #16252 still open), but trajectory is clear.

### Other notable v1.3.13 features

- **`bun install` 17x memory reduction** via tarball streaming. Widens perf gap vs `pnpm install`. Bun-on-Nx pitch gets stronger if lockfile gets fixed.
- **5.5x faster gzip via zlib-ng**. Affects pkg install + cached artifact decompression.
- **SHA3 in node:crypto + WebCrypto**. Node compat tightens.
- **Range request support in `Bun.serve()`**. Relevant for serving large artifacts (Nx Cloud-style).

### Nx Compatibility Assessment

- **Lockfile parsing (bun#16252)**: **STILL OPEN, unassigned, no PR.** Verified live 2026-05-04. Persistent blocker.
- **Node compat**: Improvements in v1.3.13 (SHA3, ws+unix, ServeStaticPlugin range). No regressions reported.
- **Adoption**: Continuing growth. Now competing in test-orchestration space.

### Risk Level for Nx: MEDIUM → MEDIUM-HIGH

Bun moving into orchestration territory increases competitive pressure on Nx CLI's test affected/sharding story. Not a tomorrow problem but trajectory shifted.
