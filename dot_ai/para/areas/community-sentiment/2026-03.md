## Key Takeaways

- **Downloads surged**: ~85.4M `nx` downloads over 60 days, averaging ~8.7M/week on full weeks. This is a significant jump -- the npm API now captures CI/mirror traffic more completely, so compare week-over-week rather than to last month's absolute numbers.
- **Angular still dominates issues**: 19 open issues mention Angular. Release/publish (19), Rspack (8), Vite/Vitest (16 combined), Maven (8), and NestJS (7) follow.
- **120 unresponded open issues** across all open issues (0 comments). 75 of these are >30 days old. This is the most actionable metric to improve.
- **168 issues opened, 72 closed** in the window = 43% resolution rate. Consistent with last scan. Many recent issues simply too new for triage.
- **14 issues resolved since last scan** (Mar 2-4), including Angular unit-test fixes, Jest v30 migration, native hasher bugs, and ESLint 10 compat.
- **Security dependency concerns emerging**: minimatch CVE (#34701), fast-glob via copy-webpack-plugin (#34632), koa CVE in module-federation (#34621) -- three separate supply chain issues reported in one week.
- **Stale high-vote debt unchanged**: Top 10 all-time voted issues remain 2-5 years old with 28-85 reactions. None resolved this period.

## Emerging Patterns

1. **Supply chain / security concerns**: Three CVE-related issues in one week (minimatch, fast-glob via copy-webpack-plugin, koa in module-federation). Users are running `npm audit` and expecting Nx to stay clean. This is a new pattern not present in last scan.

2. **AI tooling surface area**: Issues around `aiAgents` option (#34692), Codex sandbox support (discussion), MCP context exposure, git worktrees. Nx's AI-first push is generating new issue categories.

3. **Bun compatibility**: 5 issues (lockfile parsing, publish hangs, TUI bugs, getCallSites [FIXED], loop detection). Bun adoption growing; Nx keeping pace but surface area expanding.

4. **Angular + Rspack + Module Federation**: This three-way intersection continues to produce the most complex bugs. HMR, lazy routes, code splitting, CSS paths all have open issues. 6 Angular issues fixed in last 3 days though -- good momentum.

5. **NestJS cluster emerging**: 7 open issues around NestJS builds (empty deps, ESM, Swagger plugin). NestJS is the second most popular Node framework and Nx support has rough edges.

6. **TUI stability**: Terminal freezes (#34654), run-many crashes (#34677, now responded), scrolling (#31226, 26 reactions), disabling TUI issues (#33328, 8 reactions). The TUI is getting daily use but has persistent rough edges.

7. **Lockfile pruning for Docker**: Multiple issues around pnpm lockfile pruning (#34655, #34337), catalogs, and transitive deps. Docker builds are a common deployment target where these break.

8. **Java/Maven ecosystem**: 8 issues. Configurable pomPath, nested modules, project name resolution, execution order all have bugs. Active non-JS ecosystem.
