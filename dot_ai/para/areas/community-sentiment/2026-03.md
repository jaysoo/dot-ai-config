# Community Sentiment — March 2026

_Scan date: 2026-03-04_
_Period: January 3 – March 4, 2026 (60-day lookback)_

## Key Takeaways

- **Downloads strong and steady**: ~39.5M `nx` downloads over 60 days, averaging ~8.5M/week (excluding partial final week). @nx/vite (14.2M) now outpaces @nx/react (10.4M) and @nx/angular (8.8M).
- **Angular dominates issues**: 22 issues mention Angular — by far the most-referenced framework. Rspack (10), Vite/Vitest (19 combined), Maven (11), and release/publish (15) follow.
- **70 unresponded open issues** in the lookback window (0 comments, still open). This is the most actionable metric to improve.
- **167 issues opened, 72 closed** in the window = 43% resolution rate. Down from the 91% rate in last scan's shorter window. Many recent issues are simply too new.
- **Module Federation remains painful**: 6 issues across Angular+Rspack+MF. The integration surface area keeps generating bugs.
- **Nx Release edge cases**: 4 open issues around independent projects, changelog generation, and package manager detection.
- **Stale high-vote debt unchanged**: Top 10 all-time voted issues are 2–5 years old with 28–85 reactions. None resolved this period.

## Emerging Patterns

1. **AI tooling surface area**: Issues around `aiAgents` option (#34692), Codex sandbox support (discussion), MCP context exposure. Nx's AI-first push is generating new issue categories.

2. **Bun compatibility**: 4 issues (lockfile parsing, publish hangs, TUI bugs, getCallSites). Bun adoption is growing and Nx needs to keep pace.

3. **Angular + Rspack + Module Federation**: This three-way intersection continues to produce the most complex bugs. HMR, lazy routes, code splitting, and remote entry all have open issues.

4. **Java/Maven ecosystem**: 11 issues. The batch executor, config reading, nested modules, and project name resolution all have bugs. This is the second most active non-JS ecosystem after Angular.

5. **Lockfile pruning for Docker**: Multiple issues around pnpm lockfile pruning (#34655, #34337), catalogs, and transitive deps. Docker builds are a common deployment target where these break.

6. **TUI stability**: Terminal freezes (#34654), run-many crashes (#34677), scrolling issues (#31226). The TUI is getting daily use but has rough edges.
