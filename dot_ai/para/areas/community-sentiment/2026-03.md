## Emerging Patterns

1. **Supply chain / security concerns**: Three CVE-related issues in one week (minimatch, fast-glob via copy-webpack-plugin, koa in module-federation). A PR (#34708) is in progress to address all three. Users are running `npm audit` and expecting Nx to stay clean.

2. **AI tooling surface area**: Issues around `aiAgents` option (#34692), Codex sandbox support, MCP context exposure, git worktrees. Polygraph commands being added to core. New issue category that did not exist 3 months ago.

3. **Module Federation + Angular + Rspack**: This three-way intersection produces the most complex bugs. HMR (#34627), lazy routes (#34476), code splitting (#34559), ESM exports (#34584), production builds (#34083 with 15 comments). Most-discussed cluster.

4. **Java/Maven ecosystem**: 12 issues. Largest non-JS cluster. Configurable pomPath, nested modules, batch implementation, execution order. Active ecosystem with dedicated users but limited team bandwidth.

5. **TUI stabilization**: #33328 (disabling TUI) closed today. Terminal freezes (#34654), run-many crashes (#34677), scrollbar positioning being fixed. Trend improving.

6. **Lockfile pruning for Docker**: Multiple issues around pnpm lockfile pruning (#34655, #34337 FIXED), catalogs, and transitive deps. Docker is a common deployment target where lockfile issues surface.

7. **nx release edge cases**: 17 open issues. Independent projects relationship, changelog for dependent libs, version plan scoping, GitHub-only releases. The feature is mature for simple cases but complex workflows hit walls.
