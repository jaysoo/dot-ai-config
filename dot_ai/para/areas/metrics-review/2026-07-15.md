# Nx organic search decline assessment - 2026-07-15

## Executive summary

- Organic traffic is reported down about 46% versus October 2025, with the main break in December.
- Google's December 2025 core update (December 11-29) is a credible hypothesis if GSC Web Search losses begin within that window.
- The February 2026 Discover update cannot explain a December Web Search decline. No December 2025 spam update was reported.
- A current nx.dev spot check found no obvious sitewide crawl block: robots, sitemaps, sample canonicals, and descriptions are present.
- Exact attribution needs GSC page/query exports around the update; broad content changes before that would be premature.

## Data needed

Search type: Web. Compare 2025-11-13 through 2025-12-10 with 2026-01-06 through 2026-02-02.

Export:

- Daily site totals.
- Pages and queries with clicks, impressions, CTR, and position.
- Countries, devices, and search appearances.
- Repeat for `/docs/`, `/blog/`, and all remaining marketing pages.
- Separate Discover report export.
- Page indexing, crawl stats, manual actions, and security report summaries.

## Decision rules

- Ranking loss: positions and impressions fall in the same page/query cohorts.
- Demand loss: impressions fall while positions remain stable.
- CTR/SERP loss: impressions and positions hold while CTR falls.
- Technical loss: indexing/crawl signals change with the traffic drop.
- Localized content problem: a small set of URLs explains most lost clicks.

## Action order

1. Technical/indexing checks.
2. Loss-concentration analysis.
3. Intent and cannibalization map.
4. Evidence-rich refresh of affected pages.
5. Internal-link and snippet improvements.
6. Measure for 4-8 weeks and across the next relevant ranking update.
