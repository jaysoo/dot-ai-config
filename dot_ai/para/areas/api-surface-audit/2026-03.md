## Patterns and Systemic Issues

1. **Generator Rename Pattern**: Many generators were renamed (e.g., `jest-project` -> `configuration`, `rollup-project` -> `configuration`) but docs still list old names. This is a systemic issue across 8+ packages.

2. **`convert-to-inferred` generators**: Nearly every plugin now has a `convert-to-inferred` generator for Crystal migration, but none are documented in packages.json. At least 12 packages affected.

3. **Angular executor modernization**: Angular moved from webpack-based to application/esbuild-based executors, but docs still list the old webpack variants.

4. **Entirely undocumented packages**: 7 packages with real generators/executors have zero documentation (docker, gradle, maven, playwright, remix, rspack, vitest).
