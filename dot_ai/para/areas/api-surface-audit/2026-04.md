1. **Generator rename drift**: `*-project` -> `configuration` across 8+ packages. Docs still list old names (jest-project, rollup-project, esbuild-project, webpack-project, cypress-project).

3. **Angular executor modernization**: Docs still list webpack-based executors (`webpack-browser`, `webpack-server`, `dev-server`) while repo has `application`, `browser-esbuild`, `unit-test`.

4. **`init` generators**: Docs list `init` as visible for many packages, but repo marks them all as `hidden` or `x-hidden`. The 44 stale generators count includes these.

5. **12 entire packages** with user-facing generators/executors have zero documentation pages.

6. **`linter` -> `eslint` rename** still not reflected. Docs have `linter` with `eslint` executor; repo has `eslint` package with `lint` executor.
