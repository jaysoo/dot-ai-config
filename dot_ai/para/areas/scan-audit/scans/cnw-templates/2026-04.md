## Action Items

### Fix ASAP (CRITICAL)

1. **axios SSRF via @module-federation (angular + react templates):** The `@module-federation/enhanced@2.3.1` bundled by `@nx/module-federation` depends on `axios` <1.15.0 which has a critical SSRF vulnerability (NO_PROXY Hostname Normalization Bypass). **7 critical vulns per template.** Options:

   - Or add `axios@^1.15.0` as an override in template package.json
   - Or remove `@nx/module-federation` from default template deps if not needed by default
