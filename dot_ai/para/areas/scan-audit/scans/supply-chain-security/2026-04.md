1. **[HIGH] Enable OIDC + provenance on ocean publishing pipeline** (carried) — `nx-cloud@19.1.3` and `@nrwl/nx-cloud@19.1.3` both publish without SLSA attestations. Biggest remaining gap in supply chain posture. Ocean repo wasn't available this scan; confirm with ocean team.

4. **[LOW] Reclaim `nrwl` and `nx-workspace` npm package names** (carried) — Both unpublished/reclaimable.
5. **[LOW] Add explicit `provenance=true` to .npmrc or release config** (carried) — Provenance IS generated via OIDC but explicit config prevents regression.
6. **[LOW] Add top-level `permissions: {}` to `ci.yml`, `publish.yml`, `generate-embeddings.yml`** (carried) — Defense-in-depth. Still not set; only job-level permissions configured.
7. **[LOW] Reduce maintainer count on `nx` core package** (carried) — 8 maintainers vs 5 for `@nx/*` scoped packages.
