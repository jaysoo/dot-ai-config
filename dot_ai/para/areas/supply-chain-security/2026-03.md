## Recent CVEs and Supply Chain Incidents

### CVE-2025-10894 (S1ngularity) -- RESOLVED

- **Date**: August 26, 2025
- **Severity**: Critical
- **Attack vector**: Exploited GitHub Actions `pull_request_target` trigger to steal npm publishing token
- **Impact**: Malicious versions of nx, @nrwl/nx, @nx/devkit, and other packages published for ~5 hours; exfiltrated cryptocurrency wallets, GitHub/npm tokens, SSH keys, environment variables; weaponized AI CLI tools (Claude, Gemini, Q) for data exfiltration
- **Remediation**: All malicious versions removed. Nx team migrated to npm Trusted Publishers (OIDC), enforced 2FA, eliminated `pull_request_target` triggers, added PR release script integrity checks
- **Current status**: Fully remediated. No `pull_request_target` triggers in any workflow. OIDC-based publishing confirmed active.
- **References**: [NVD](https://nvd.nist.gov/vuln/detail/CVE-2025-10894), [ZeroPath Technical Review](https://zeropath.com/blog/cve-2025-10894-nx-npm-supply-chain-attack-summary), [Nx Postmortem](https://nx.dev/blog/s1ngularity-postmortem), [CISA Alert](https://www.cisa.gov/news-events/alerts/2025/09/23/widespread-supply-chain-compromise-impacting-npm-ecosystem)

### Broader npm Ecosystem Incidents (2025-2026)

1. **Shai-Hulud Worm (Sep 2025)**: Self-replicating worm compromised 500+ npm packages including `debug` and `chalk`. Root cause was phished maintainer 2FA. npm responded with authentication overhaul: revoked all classic tokens, defaulted to session-based tokens.
2. **npm Authentication Overhaul (Dec 2025)**: npm deprecated classic tokens, required granular access tokens (7-day expiry for publishing), deprecated TOTP 2FA in favor of FIDO-based 2FA.
3. **Ongoing threats (2026)**: npm ecosystem continues to face typosquatting and dependency confusion attacks. No new incidents directly affecting Nx packages.

**References**: [npm Hardening Update (Feb 2026)](https://thehackernews.com/2026/02/npms-update-to-harden-their-supply.html), [CISA npm Alert](https://www.cisa.gov/news-events/alerts/2025/09/23/widespread-supply-chain-compromise-impacting-npm-ecosystem), [npm Supply Chain Defense Guide 2026](https://bastion.tech/blog/npm-supply-chain-attacks-2026-saas-security-guide)

No new CVEs affecting Nx packages found in 2026.
