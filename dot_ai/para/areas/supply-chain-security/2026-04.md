**Assessment:** Zero critical vulnerabilities. The +47 increase is from new advisories against existing transitive dependencies (fast-xml-parser, path-to-regexp, body-parser, multer, astro allowlist bypass). None affect the core `nx` package, `@nx/devkit`, or the CLI binary. All high-severity items remain in: (1) docs site tooling, (2) framework plugin dev dependencies, (3) test tooling. The `fast-xml-parser` advisory is new and worth monitoring since it appears as a direct dependency.
