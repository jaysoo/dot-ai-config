### Key High-Severity Packages (43 unique, risk-classified)

Same full list as 2026-04-13 scan. Three warrant closer inspection:

1. **fast-xml-parser** (CVE-2026-26278 entity expansion bypass) — **Confirmed devDependency only** in root `package.json` (`^5.3.7`). Not pulled into any published `@nx/*` package. Risk: LOW.
2. **rollup** (arbitrary file write via path traversal) — `packages/rollup/package.json` declares `rollup: ^4.14.0` as a plugin peer/dep. Vulnerable range is `>=4.0.0 <4.34.8`. Users of `@nx/rollup` inherit this. Verify in next scan whether installed range has moved past 4.34.8 or whether we need to bump the peer range.
3. **serialize-javascript** (RCE via RegExp.flags) — dev/build tool dep (webpack/rollup plugin); not bundled into published packages.
