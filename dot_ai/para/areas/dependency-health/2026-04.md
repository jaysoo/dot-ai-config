## Recommended Actions

### Priority 1 -- Immediate attention (Critical + High Impact)

1. **`enquirer`** (Critical, 9 packages) -- **MOST URGENT**. Last publish Jul 2023 (33mo). Core interactive prompts across Nx. Evaluate `@inquirer/prompts` migration. Unchanged since March.

2. **`source-map-support`** (Critical, @nx/js) -- Node 20+ has built-in `--enable-source-maps`. Drop this dep for supported Node versions.

3. **`tcp-port-used`** (Critical, @nx/node) -- Replace with `detect-port` which Nx already uses in @nx/cypress, @nx/js, @nx/web.

4. **`deepmerge`** (Critical, @nx/angular-rspack) -- Consider `deepmerge-ts` (typed, maintained) or `lodash.merge`.

### Priority 2 -- Consolidation opportunities

5. **Terminal coloring**: `ansi-colors` (Critical), `chalk` (Healthy), `picocolors` (Warning 14 pkgs). Consolidate to `picocolors` -- already most widely used, smallest footprint.

6. **`webpack-subresource-integrity`** (Critical) -- Still on RC (5.2.0-rc.1) after 3+ years. Evaluate if SRI is still needed or find alternative.

7. **`babel-plugin-const-enum`** + **`babel-plugin-macros`** (both Critical) -- Evaluate if needed with modern TypeScript / SWC.

### Priority 3 -- Monitor closely

8. **`picocolors`** (Warning, 14 packages) -- Highest impact Warning package. Stable but single maintainer.

9. **`rxjs`** (Warning, @nx/webpack) -- v7.8.2 last published Feb 2025. Monitor Angular's stance on rxjs v8.

10. **`yargs`** (Watch, 3 packages) -- Core CLI framework. Last publish May 2025. Mature but aging.

11. **`loader-utils`** (Warning) -- Had critical CVE (CVE-2022-37601) in v2. Verify all usages are on v3.x.

12. **`@rspack/plugin-react-refresh`** (Healthy) -- Major version bump to 2.0.0 on Apr 8. Verify Nx rspack plugin compatibility.

### Priority 4 -- No action (stable but old)

These Critical packages are stable, low-level utilities with no functional issues:
- `tree-kill`, `identity-obj-proxy`, `concat-with-sourcemaps`, `license-webpack-plugin`
- `flat`, `lines-and-columns`, `columnify`, `@rollup/plugin-image`, `@rollup/plugin-json`
- `source-map-loader`, `@ampproject/remapping`, `@svgr/webpack`, `figures`
