## Recommended Actions

### Priority 1 -- Immediate attention (Critical + High Impact)

1. **`enquirer`** (Critical, 9 packages) -- **MOST URGENT**. Last publish Jul 2023 (33mo). Core interactive prompts across Nx. Evaluate `@inquirer/prompts` migration. Unchanged since last report.

2. **`source-map-support`** (Critical, @nx/js) -- Node 20+ has built-in `--enable-source-maps`. Drop this dep for supported Node versions.

3. **`tcp-port-used`** (Critical, @nx/node) -- Replace with `detect-port` which Nx already uses in @nx/cypress, @nx/js, @nx/web.

4. **`@ltd/j-toml`** (Critical, nx) -- Replace with `smol-toml` (actively maintained, last publish Mar 23 2026). Note: `smol-toml` is already in the Nx dependency list.

5. **`deepmerge`** (Critical, @nx/angular-rspack) -- Consider `deepmerge-ts` (typed, maintained) or `lodash.merge`.

### Priority 2 -- Consolidation opportunities

6. **Terminal coloring**: `ansi-colors` (Critical), `chalk` (Healthy), `picocolors` (Warning). Consolidate to `picocolors` -- already most widely used (14 packages), smallest footprint.

7. **`webpack-subresource-integrity`** (Critical) -- Still on RC (5.2.0-rc.1) after 3+ years. Evaluate if this feature is still needed or if a webpack plugin alternative exists.

8. **`babel-plugin-const-enum`** + **`babel-plugin-macros`** (both Critical) -- Evaluate if these Babel plugins are still needed with modern TypeScript / SWC.

### Priority 3 -- Monitor closely

9. **`picocolors`** (Warning, 14 packages) -- Highest impact Warning package. Stable but single maintainer. No action needed yet.

10. **`rxjs`** (Warning, @nx/angular, @nx/webpack) -- v7.8.2 last published Feb 2025. rxjs v8 has been in discussion. Monitor Angular's stance.

11. **`yargs`** (Watch, 3 packages) -- Core CLI framework. Last publish May 2025. yargs ecosystem is mature but aging. Long-term, consider alternatives if it goes stale.

12. **`loader-utils`** (Warning) -- Had critical CVE (CVE-2022-37601) in v2. Verify all usages are on v3.x.

13. **`fork-ts-checker-webpack-plugin`** (Warning) -- Only 6 days from becoming Watch-eligible. Published Apr 3, 2025. Monitor.

14. **`@rspack/plugin-react-refresh`** (Healthy) -- Major version bump to 2.0.0 on Apr 8. Verify Nx rspack plugin compatibility.

### Priority 4 -- No action (stable but old)

These Critical packages are stable, low-level utilities with no functional issues:
- `tree-kill`, `identity-obj-proxy`, `concat-with-sourcemaps`, `license-webpack-plugin`
- `flat`, `lines-and-columns`, `columnify`, `@rollup/plugin-image`, `@rollup/plugin-json`
- `source-map-loader`, `@ampproject/remapping`, `@svgr/webpack`

## Trend Analysis

The distribution is stable compared to the April 7 snapshot. The only movement was `tinyglobby` graduating from Watch to Healthy after receiving a patch release. All Critical and Warning packages remain stagnant -- none received updates in the past 2 days.

**Key risk vector**: The webpack ecosystem (loaders, plugins) continues to age collectively. As Nx invests more in Rspack/Vite, the webpack-specific deps will naturally become less urgent, but they still need maintenance for existing users.

**Positive signals**:
- `@phenomnomnominal/tsquery` remains actively maintained (6.2.0, Apr 6)
- `tinyglobby` received a timely update, keeping it healthy
- `@rspack/plugin-react-refresh` hit 2.0.0, showing active Rspack ecosystem investment
- Core build tools (webpack, vite, esbuild, rollup) all received updates this week

**Approaching boundaries**:
- `fork-ts-checker-webpack-plugin` (Apr 3, 2025) will cross into Watch territory on Apr 9
- `confusing-browser-globals` (Feb 14, 2025) will age into Warning->Watch in Jul 2025
