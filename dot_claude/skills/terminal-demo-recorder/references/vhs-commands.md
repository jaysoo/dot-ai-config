# VHS Commands Reference

## Core Commands
- `Output <path>`: Specify the output file (e.g., `Output demo.gif`, `Output demo.mp4`).
- `Set <Setting> <Value>`: Configure the recording.
- `Type "<text>"`: Type text into the terminal.
- `Enter`: Press the Enter key.
- `Sleep <duration>`: Pause the recording (e.g., `Sleep 500ms`, `Sleep 2s`).
- `Backspace [n]`: Press the Backspace key `n` times.
- `Tab`: Press the Tab key.
- `Ctrl+<key>`: Send a control sequence.

## Settings
- `FontSize <number>`: Font size in pixels.
- `FontFamily "<name>"`: Font family name.
- `Width <number>`: Window width in pixels.
- `Height <number>`: Window height in pixels.
- `Padding <number>`: Window padding in pixels.
- `TypingSpeed <duration>`: Delay between keystrokes (e.g., `Set TypingSpeed 50ms`).
- `Theme "<name>"`: Terminal theme (e.g., `"Dracula"`, `"Monokai Pro"`).
- `LoopOffset <percentage>`: Where to start the loop in the GIF (e.g., `Set LoopOffset 0%`).
- `PlaybackSpeed <number>`: Speed multiplier (e.g., `Set PlaybackSpeed 0.5`).

## Keys
- `Up`, `Down`, `Left`, `Right`
- `PageUp`, `PageDown`
- `Home`, `End`
- `Escape`
